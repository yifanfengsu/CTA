## Q2 — 持有期是否解决了上次的死因（G2 不单调、G5 衰减）

**两个原死因都被拉长持有期解决了——假设在这两点上被证实：**
- **G2 单调性回来了**：quintile 违反 N=1 **2 → N=3 1 → N=5 1**（≤1 即过）。日频下中段
  Q3/Q4 塌陷，3-5 日持有后单调性恢复。
- **G5 近年 IC 起来了**：2025+ IC 的 t 值 N=1 **0.70 → N=3 2.27 → N=5 3.55**——单调强化，
  carry 的近年信息在更长视野下显著（2026 IC：N=1 +0.016 / N=5 +0.047）。

机制假设（funding 慢变量→数日持有）在 IC、单调性、近年稳定三个维度被证实。**但代价是
G4 反向走弱**：alpha_t N=1 2.13 → N=3 2.06 → N=5 1.90。原因（Q3）：拉长持有 + 重叠口径把
同一段总收益摊到更多天，rank IC（排序预测力）增强，但日度价差均值被稀释 → 日收益 t/Sharpe
反而下行。**IC 强化与可交易日度 alpha 弱化在持有期上反向**——这是 N=5 翻车的机制根源。

## Q3 — 双持有期一致性 + 持有期曲线（k=1/3/5）

- **一致性：矛盾。** N=3 全过、N=5 死 G4 → 结论相反 → 预注册③判**参数尖峰、可疑、不立项**。
- **持有期曲线**（k=1 锚 / 3 / 5）：
  - IC 均值：+0.0188 → +0.0312 → +0.0350，**horizon 单调增（True）**——真持有期效应，非跳变。
  - quintile 违反：2 → 1 → 1（改善后走平）。
  - 净 Sharpe：0.63 → 0.63 → **0.55**（走平后在 5 日回落）。
  - alpha_t：2.13 → 2.06 → **1.90**（单调下行，5 日跌破 2.0 门）。
- **IC 曲线单调向上 vs alpha_t/Sharpe 曲线单调向下**——信号的"排序预测力"随视野增强是真的，
  但"可交易日度经济显著性"随视野衰减。立项要的是后者（扣成本 Sharpe / alpha_t），它在 N=5
  失守。一致性裁决因此判矛盾：不是 IC 不稳，是 IC 强化没能转化为稳定的可交易 alpha。
- **N=1 复现校验**：IC +0.0188（上次 +0.0188，逐字一致）、净 Sharpe 0.63（上次 ~0.61，
  微差源于 CAR 专用预热起点 imin=8 vs 上次全因子统一 31，多纳入约 4 个早期低币日，不重要）。
  **引擎复现确认。**

## Q4 — 成本变化（重叠持仓使换手/成本降低）

重叠持仓如预期降低换手与成本（成本/毛利）：**N=1 1.50% → N=3 1.10% → N=5 0.93%。**
日成本按重叠组合净权重变化 Σ|Δw|·fee 计，重叠使日度净权重变化 ~1/N。**但成本从来不是
binding 约束**（三个持有期成本占比都 <2%）；carry 的生死线在 G4 alpha 显著性，不在成本。

## Q5 — 立项判定

**不立项。** 双持有期矛盾（N=3 全过\* / N=5 死 G4）= 预注册③的参数尖峰 → 可疑 → 不立项。
未触发立项算术（无稳健通过）。

**carry 这条 MR 线索在幸存者样本上的状态——如实、分层记录（不强塞单一结论）：**
1. **机制假设被部分证实**：拉长持有期确实解决了 k=1 的两个死因（不单调、近年衰减），
   IC 随 horizon 单调增强。"carry 被错误的日频持有期埋没"在 IC/单调/近年三维上成立。
2. **但未能转化为稳健可交易 alpha**：IC 强化的同时日度 alpha_t 单调走弱，N=5 跌破 2.0；
   3 日过、5 日败 = 参数敏感。立项要的是扣成本可交易显著性，它不稳。
3. **纪律裁决优先于"差一点"**：N=5 的 1.90 差 0.10，但纪律②禁止放宽 2.0 门、③定义一过一败
   为可疑。放宽即重蹈 1.5×DD gate 差 0.023 的覆辙——不放。**故不立项。**
4. **重启门槛已抬高**：carry 已被事后调整两次（先因子本身、再持有期）。任何后续重看须用
   **真正的 OOS 杠杆**——补退市币的幸存者校正样本 + 单一预注册持有期——而非第三次参数探查。
   记录不行动。

## Q6 — 观察节

- **本研究是"纪律拦截参数尖峰"的清晰实例**：carry 在 3 日持有下全过 5 门、看着像 alpha，
  但 5 日就因 alpha_t 跌破 2.0 翻车。若只跑 3 日、不跑 5 日做稳健性对照，会误立项。
  双持有期一致性检验（预注册③）正是为此设计——它拦住了。
- **IC ≠ 可交易 alpha 的又一次实证**：rank IC 随持有期单调增强（+0.019→+0.035）极漂亮，
  但摊薄到日度后 Sharpe/alpha_t 反而下行。**排序预测力强 ≠ 扣成本经济显著**——与上次横截面
  研究 MOM"IC≈0 但价差好看"互为镜像（这次是"IC 强但日度 alpha 弱"）。
- **carry 是整个横截面里唯一两次都留痕的信号**：显著 IC + 真 beta-中性 alpha + 机制可解释 +
  近年 IC 在长持有下显著。它没立项，但它是这片版图里"最像有东西"的地方——若未来项目接入
  退市币做幸存者校正、或进入可容忍长持有的多线阶段，carry 是第一顺位重看对象（单一预注册
  持有期、勿再扫描）。**记录不行动。**
- **beta 中性再次确认**：三个持有期价差 beta 都 −0.10~−0.14、两腿 beta 1.04-1.18 近对称——
  横截面构造稳定地对冲掉方向暴露。先验担心的"伪装 beta"在 carry 上同样被证否。
