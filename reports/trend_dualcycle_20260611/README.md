## Q3. "前载"终审：是结构缺陷，不是 OKX 窗口巧合

六年数据把问题暴露得更彻底：慢配置（B1 50/200、C2 90d、全部 D-慢速变体）的
V5 死因年份从"2023"换成了 **2020（占比 0.83-0.88）**。机制实锤：B1_4h 的单笔
DOGE 多单 **2020-11-08 入场 → 2021-05-24 出场，净 $1,012,799，占其六年总净利
81%**——慢趋势配置每个周期本质上只持有一张跨年彩票，V5 按入场年归因时整段
利润全部落在入场年。**"利润前载到最大趋势事件"是这类配置的结构属性，
换任何含抛物线的样本都会再现**；D1/D3/D4 的 OKX 死刑维持原判。
快配置（B2 20/100 / D2）中途换手再入场，利润摊到 2020(19-20%) + 2021(55-58%) +
2023(15-18%) + 2024(9-14%)，两个样本上都过 V5——**快慢之差是真正的分界线**。
如实记录张力：V5 惩罚的恰是 V1' 认证为"结构性尾部收割"的东西；gate 体系
对慢趋势天然不利，这是体系属性，本次不动任何 gate，仅记录。

## Q6. 综合：最终候选池 = 一个信号结构的两种风险形态

双周期幸存者：**B2_4h（EMA20/100 long/short）与 D2（同信号 long/flat）**。
组合阶段已证明 D2 持仓是 B2_4h 多头腿的严格子集（不应同收），故最终候选池
实质是**一个信号结构（4h EMA20/100）的两种形态**，二者各自在两份独立样本上
全 gate 过堂；2022 深熊证据（Q2）支持含 short 腿的 B2_4h 为首选形态。
趋势线没有全军覆没，但也必须如实说：**候选从"5 个配置"收敛到"1 个信号"**，
与组合阶段"有效独立信号数 ≈2（EMA 簇 + TSMOM 簇）"中 TSMOM 簇双周期落选
（C2_4h/C2_1d 均死 V5）的结果互相印证。星号永久保留（V1' 系事后修正检验 +
幸存池系全样本选择产物）。前向验证候选据此更新：单 sleeve B2_4h（或其与
慢腿组合的问题留给用户决策）；组合阶段对 2023+ 样本显著性的否定结论不变。
