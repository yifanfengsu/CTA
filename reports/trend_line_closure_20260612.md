## 4. 方法论教训（本线最大产出）

**验证周期必须是选题维度。**

低频弱 edge 信号的统计验证以年计，且这在开题日即可估算：年信号笔数 × 预期
Sharpe → 统计判别所需样本长度（量级公式：排除 0 所需年数 ≈ (1.96 / 年化
Sharpe)²；Sharpe 0.5 → ~15 年，与组合阶段实测的 15–19 年一致）。趋势线在
开题时即可知：4h 级趋势信号、文献先验 Sharpe ~0.5，则 3.4 年样本永远不够、
前向验证必然以年计。这个算术却在五个阶段、五份报告之后才成为决策因素——
五个阶段本身执行纪律无可指摘（预注册、零调参、双样本复测），但它们回答的
都是"信号是否稳健"，没有一个阶段回答"验证这件事本身是否负担得起"。

已固化为 CLAUDE.md 工作原则：任何新策略线开题时必须估算验证周期并确认与
可接受资源配置匹配后才立项；验证周期超出可接受范围的方向，无论形态多有
吸引力，不立项。
