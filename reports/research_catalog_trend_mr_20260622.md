### 1.3 趋势线关闭：资源决策，**非证伪**【已核对 trend_line_closure】

关闭日两列证据**同时为真**：

**支持面**：六年双样本双交易所全 gate 存活（DUAL-VALIDATED\*）；2022 深熊
short 腿正赔付；6y bootstrap CI 排除 0；跨所偏差仅 +2%。

**关闭理由**：Sharpe ~0.5 量级信号的统计验证周期与项目可接受资源配置不匹配。
验证周期数学（CLAUDE.md 铁律）：

> 验证年数 ≈ (1.96 / 年化 Sharpe)²；Sharpe 0.5 → **~15.4 年**（与组合阶段
> 实测 15–19 年一致）；前向部分验证**18 个月起步**。

**重启条件（留门，非证伪）**：① 第二信号族产出与趋势低相关的幸存者，使组合
ENB 提升（当前池 ENB 仅 1.72，聚合无法救显著性）；② 项目进入"多线并行、
可容忍长周期观察"阶段。

### 2.1 MR-5m 原始策略 + 污染事故

#### 策略本体

5m Donchian 通道反向（fade）策略，全套工程落地（vnpy 回测模板 + 独连 OKX
实盘 runner：IOC 下单 / 仓位对账 / force_close / bills 对账 / 微信通知），

#### 污染取证 — `reports/regime/data_contamination_forensics_20260610/`【已核对】

- **污染机制**：`.env` 第 4 行 `OKX_SERVER=DEMO` → `download_okx_history.py`
  默认 gateway 路径全程带 `x-simulated-trading: 1` → OKX 返回**模拟盘环境**
  K 线。当前库 5 币 × **1,791,360 根 1m**（每币）**全部**经此路径写入。
- 唯一干净的代码路径（mainnet 公共 REST 备胎）**从未触发**（0 次 fallback）。
- 598 个合成 ramp 事件（**SOL 344 / DOGE 216 / ETH 23 / LINK 15 / BTC 0**）
  只是 demo 行情偏离最剧烈、**可被检出的子集**——全库逐 bar 偏差程度更广。
- 旁证：`reports/regime/v2b_dd_diagnosis_20260610/` 是发现链的引爆点——v2B
  的 maxDD 元凶诊断为 2025-05-29 的 4 笔交易 fade 了合成直线 ramp（数据
  artifact，非市场模式），由此扫出全期 598 起异常。

#### demo vs mainnet 对比 — `reports/regime/demo_vs_mainnet_comparison_20260610/`【已核对】

- **205 个币种×月份格中 199 个 red（97%）**；任何合理阈值下结论方向不变。
- 信号重合率（Jaccard）仅 **0.34–0.55**——约一半 demo 信号在真实数据上不
  存在，约一半真实信号 demo 里没有。
- demo Wilder ATR 系统性偏高（BTC 中位 **+36.7%**，90% 的 bar demo>main）。
- 归因：单笔均值 **+$8.88 → −$1.09**（−$9.98/笔）、胜率 **48.3% → 36.1%**；
  **demo 毛利 edge 的 ~99% 是数据假象**。
- 时间结构：2025-07→2026-03 持续脱锚 + 2026 年 Jaccard 跌至 0.17–0.31——
  离现在越近的数据反而越不可信。

#### mainnet 干净基线终审 — `reports/regime/mr5m_mainnet_baseline_20260611/`【已核对】

- **MR-5m FLAT 在真实数据上无 edge。** 全期 PF **0.81–0.85**、test 期 PF
  0.81–0.83；**毛利≈0**（全期 +$6k~$10k，$50M+ 成交额上是噪声），净亏 100%
  来自手续费；**无任何 币种×年份 PF>1**（最高 SOL 2023 = 0.99）。C2 配置
  全期净 **−$66,894**。
- demo 的 FLAT PF 2.06 / +$189k test 净利系数据假象。
- 机制叙事不重现：① "2024 是全市场 MR 逆风年"不重现（各年 PF 均匀差，
  2023 反而最高）；② "whipsaw 是敌人 / 健康月 97%"前提不存在（无健康月）；

## §3 相关波动率研究（独立类别，信号空间闭环一环）【已核对】

`reports/volatility_event_20260613/`——低波状态后做多波动率（1h 单尺度，
k∈{6,12,24,48}h）。

- **可预测性：双样本 PASS**（波动率聚集物理基础充分）：低波后 RV 全 < 无条件、
  高波后全 > 无条件，5/5 币 × 2 样本逐格复现。

  高波概率仅 **1.1%** vs 无条件 24.8%）——低波预测的是"平静延续"，不是
  "爆发临近"。

## §4 综合结论【已核对，引用既有闭环文档】

### 右偏延续结构（项目最持久的机制认知）

这个市场（5 币永续，中低频）是**右偏延续结构**——均值回归在**单序列 / 横截面 /
价差**三种正交构造下**三次显形**、三次被同一结构杀死：

- **单序列**（mr_timescale / mr5m_baseline / funding_structure）：回归作为频率
  现象真实存在，但少数延续（趋势）尾部移动远大于多数小回归，把逆势期望整体
  吃成负。
- **横截面**（cross_sectional_ic）：反转因子 F-REV 正 IC 是假象，可交易价差
  强负（净 −159%，"赢家继续赢"）。
- **价差**（pairs_cointegration）：|z|≥2 后回归率 17.9% < 无条件 42.4%
  （z−13.1）；偏离更多是协整关系断裂的延续，价差本身非平稳。

三切面同源：**逆势负期望、顺势弱 edge、波动率无载体**。顺势（趋势 B2_4h）
收割的正是同一条右偏尾部，所以它能存活；但 Sharpe ~0.5 在几年样本下数学上
不可能显著。波动率高度可预测，但可预测的是"平静/动荡各自延续"，对线性永续
不可收割。

## §5 方法论资产（最重要的一节）【已核对各报告 + PROJECT_GUIDE】

### 5.1 预注册 + 双样本 + 机制验证的防线栈：判死记录

整个项目真正可迁移的产出是这套防线栈。它在多次研究中正确判死，各自拦截了
不同的失败模式：

| 拦截的失败模式 | 在哪些研究中判死【已核对】 | 拦截了什么 |
|---|---|---|
| **描线**（line-drawing）| funding 同向确认 | OKX 单边全面改善（Sharpe 0.65→0.99），Binance 6 年完全推翻——只在能看见答案的样本里改善 |
| **范畴错误**（category error）| V1 剔 top5%（r1）、ADX 强度过滤 | 对"利润分布设计上即为尾部收割"的趋势策略，用"去尾/强度确认"删的正是利润本身（ADX：~70% 头部利润在被过滤的早期低 ADX 入场）|
| **参数尖峰**（parameter spike）| carry 持有期 3日/5日矛盾、mr_timescale 孤立尖峰规则 | 3 日过 / 5 日败 = 不稳；单/双币撑起的汇总格（E2·4h·k=8）按规则不入算术 |
| **确认偏误**（confirmation bias）| 四次已知靶子增强统一防线 | 已知历史靶子上的增强，单样本证据权重 ≈ 0 |
| **条件厚度陷阱** | pairs C4（+13.8% 但 17.9% 命中）、mr_timescale（median 过线但 mean 负）、volatility（厚度过但增量负）| median 厚度过 ≠ 可行，必须看 mean / 命中率 / 增量 |
| **IC ≠ 可交易 alpha** | cross_sectional_ic、carry_holding | MOM IC≈0 但价差好看（尾部+幸存者镜像）；carry IC 随 horizon 增但日度 alpha 反降 |
| **减法/加法陷阱** | funding（减 46% 笔→周期延长）、ADX（减 56%→∞）、faster-entry（加 180% 笔→周期 26.8y）| 过滤/加信号须用**频率调整后的验证周期**记账，而非只看 per-trade 指标 |
| **"少交易让指标变好"** | MR-5m 熔断器（74 配置全败，归档）、ATR filter（只能少亏）| 减仓让指标好看是陷阱（looking better by trading less）|

### 5.2 四次已知靶子增强全败的同构教训

funding 确认（描线）/ ADX 过滤（双样本反向、范畴错误）/ faster-entry（对称
劣化）/ V1 剔 top5%（范畴错误）——四次在已知靶子上的增强**全部判死**，且都被
同一防线栈识破。**核心元教训**：

> 对已知历史的策略做增强，几乎必然描线或范畴错误；真实改善只能来自独立的
> 新信号，而非更好的过滤 / 换参数 / 晚确认。"OKX 单边好看"或"机制部分为真"
> 都骗不过双样本 + 机制独立验证 + 频率调整周期三防线。

### 5.3 其他可复用原则【已核对】

- **验证周期是选题维度**（topic selection），非事后发现：`验证年数 ≈
  (1.96/Sharpe)²`。趋势线教训固化进 CLAUDE.md——验证周期超出可接受范围的
  方向，无论形态多有吸引力，不立项。本铁律在 mr_timescale / breakout 两次
  开题前执行，半天内终结整类方向。
- **数据纯净 > 数据好看**：DEMO 污染抹掉 13 个月。铁律——唯一可信源显式锁定、
  写库脚本启动打印数据环境、新数据须与外部独立源逐 bar 交叉验证（mainnet 库
  经 Binance 205 月度文件全量交叉验证方采信）。
- **探明同等价值于立项**：否定性结论与肯定性结论同等固化、同等证据标准
  （1.5×DD gate 差 0.023 不放宽 → 整条数据事故发现链的起点；carry G4 t1.90
  不放宽到 2.0）。本存档的全部价值正在于此。
- **IC ≠ 可交易 alpha**：因子排序预测力强 ≠ 扣成本日度经济显著。
- **半衰期/中位厚度 ≠ 回归概率/可行性**：回归作为频率现象真实，但 mean / 命中率
  才决定可货币化（mr_timescale median 过线而 mean 负的核心教训）。
- **幸存者偏差方向已知（偏乐观）**：横截面/配对用 22 币幸存者样本，补退市币
  只会更差 → **否定性结论更稳**。
