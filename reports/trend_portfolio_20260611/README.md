## Q1. 相关结构：5 个配置 ≈ 2 个真实信号

日收益 Pearson 相关（窗口自 2023-04-01 共同上线日起，n=1,245 天）：

|  | B1_4h | B2_4h | C2_4h | C2_1d | D2 |
|---|---|---|---|---|---|
| B1_4h | 1 | .823 | .638 | .635 | .768 |
| B2_4h |  | 1 | .441 | .434 | .786 |
| C2_4h |  |  | 1 | **.993** | .593 |
| C2_1d |  |  |  | 1 | .588 |
| D2 |  |  |  |  | 1 |

月度口径对照同构且更高（B 簇内 0.86-0.92，C 簇内 0.994），见 `correlation/monthly_corr.json`。

- **(R3) 特征值分解**：λ = [3.68, 0.94, 0.24, 0.13, 0.01]，PC1 解释 73.7%，
  PC1+PC2 = 92.4% → **解释 80% 方差只需 2 个主成分**；有效分散数 ENB
  (Σλ)²/Σλ² = **1.72**。
- 结构一目了然：**EMA-cross 簇**（B1/B2/D2，互相 0.77-0.82）+ **TSMOM-90d 簇**
  （C2_4h/C2_1d，互相 0.993——4h 与 1d 重采样同一动量信号，实为一个配置）。
  两簇之间约 0.44-0.64。
- **(R2) 持仓重叠**（同币同向时间占比，4h×币种槽位全期分母）：
  - **C2_4h vs C2_1d：90.7%**（both-in 时 97.8% 同向）——确认实质同一信号。
  - **B2_4h vs D2：48.2% 同向、0.0% 反向、both-in 时 100% 同向**——D2（B2 long/flat）
    的持仓是 B2 多头腿的严格子集，不是新信号。
  - 跨簇（B vs C）：同向 60-67%，反向 26-33%——这 ~30% 的反向持仓时间
    是组合内唯一真实的对冲来源。

**Q1 答案：5 个配置的真实分散度 ≈ 2 个独立信号**（EMA-cross 与 TSMOM-90d），
其余是同一信号的重采样/裁剪。组合层的"分散"上限从一开始就只有这么多。

## Q3. 中心问题：组合聚合救回显著性了吗？——**没有**

单配置层 per-trade bootstrap CI 全部含 0（r1/r2 存档：B1_4h [−95,+625]、
B2_4h [−26,+354]、C2_4h [−43,+287]、C2_1d [−124,+598]、D2 [−3.4,+719]）。
本阶段换到日收益口径并聚合 4-5 个 sleeve 后：

| 组合 | 日均净收益 | 日波动 | t 值 | 95% CI | 年化 Sharpe |
|---|---|---|---|---|---|
| P-A | $43.2 | $1,712 | 0.89 | [−51.1, +138.9] | 0.48 |
| P-B | $46.4 | $1,596 | 1.03 | [−40.8, +135.1] | 0.56 |
| P-C | $44.4 | $1,708 | 0.92 | [−49.9, +139.4] | 0.50 |
| P-D | $44.0 | $1,661 | 0.94 | [−46.6, +136.7] | 0.51 |

聚合确实有一点帮助（t 从单 sleeve 的 ~0.7-0.9 升到 0.9-1.0，P-B 最高），但
**距 1.96 还差一半以上**。原因在 Q1 已给出：聚合 5 个东西实际只聚合了 ~1.7 个
独立信号，方差缩减天花板极低。算术上：年化 Sharpe ≈ 0.5 的策略要在 95% 置信度
下排除 0，需要 ~15-19 年数据；3.4 年样本在数学上不可能过这道 gate，除非真实
edge 远大于现在看到的。**这不是"再换一种组合方式"能解决的问题。**

## Q4. P-B vs P-A/P-C：B2+D2 同收是增益还是伪分散？——伪分散（β 倾斜）

- 信号层裁决：D2 持仓是 B2 多头腿的**严格子集**（同向 48.2%、反向 0%、
  both-in 同向率 100%）；加入 D2 不增加任何独立信号（ENB 不变）。
- P-B 的表面改善（net/DD 1.78 vs P-A 1.49、日波动 $1,596 vs $1,712）来自两个
  机械效应：① D2 空仓 52% 时间，稀释组合波动；② 等权下 EMA 多头腿权重翻倍，
  组合净多敞口均值从 +$1.2k 升到 +$5.8k、corr(B&H) 从 0.562 升到 0.592——
  **是往多头 β 上倾斜，不是分散**。
- P-C（D2 替换 B2，丢掉 B2 空头腿）给出反向证据：corr(B&H) 升到 0.644（四者最高）、
  2025 年变成 −$2,462（四者最差）、2024 占比超线挂掉 P2——印证 r2 阶段的结论
  "**B2 空头腿是 2025 年的时间分散器**"，扔掉它组合更像 B&H 也更脆。
- 结论：信号加倍无增益；若未来进前向验证，**B2_4h 与 D2 不应同收**，
  二选一时 B2_4h（保留空头腿）优于 D2。

## Q5. 最终候选：**0 个组合获得前向验证资格（按预注册 gate）**

如实写明：**趋势线在组合层仍无统计支持**。单配置层 5 个 VALIDATED* 的形态
（结构性尾部、时间稳健、邻域平坦、抗滑点）在组合层完整保留（P2/P3 轻松通过，
滚动 12 月 87-93% 为正），但 3.4 年样本下日收益均值不能与 0 区分（P1 全灭），
且 2025 至今近 18 个月无利润。剩余选项（陈述，不替用户决策）：

1. **接受终局**：趋势组合证据等级到此为止，项目转向其他策略族。3.4 年数据
   对 Sharpe ~0.5 的策略族就是不够长，这是数学事实而非执行失误。
2. **零成本前向观察**（不部署资金）：按 Q6 形态对 P-A 或 P-D 做本地信号生成 +
   模拟记账。前向样本与全样本选择独立，是唯一能在不增加回测自由度的前提下
   积累新证据的途径；但按年化 Sharpe 0.5 估算，前向期也需要数年才有判别力，
   须以"低成本长期挂着"的心态对待。
3. **降低显著性要求改用经济判据**——明确不推荐：这等于事后改 gate，
   是本项目纪律红线。

若选 2，候选结构按本阶段数据为 **P-D**（年份分布最均匀、2025 唯一为正、滚动
正占比最高 93%、corrBH 最低 0.536）或 **P-A**（无任何权重自由度，证据最干净）；
P-B/P-C 因 Q4 的 β 倾斜证据不建议。

- 预注册纪律的代价与价值在本阶段都很清楚：P-C 的 2024 占比 0.619 距线 0.019、

- 工程自检全绿：5 个 sleeve 的 M2M 总账与 trade 总账差额 $0.00；组合逐 bar 与

## 附录：方法细节

- 日度化：per-bar M2M PnL（r2.m2m_pnl 原样复用，funding 记入结算 bar）按 bar
  结束时刻 `ceil('D')` 桶到 UTC 00:00 日界；4h 与 1d sleeve 严格同界对齐。
- 相关窗口自共同上线日 2023-04-01（最晚 sleeve = C2_1d 的 90d 预热）起，避免
  预热期共同零值污染相关估计；组合记账与 gate 用全期。
- 敞口口径：$10k 名义/持仓信号 × 权重（整数张数的真实名义与 $10k 差异 <1%，
  不影响分布形状）。
- 滚动 12 月窗口 = tv.ROLL_STARTS（2023-01..2025-06 起点，30 窗），与 V5 同口径。
- 产物：`correlation/`（daily/monthly 相关、overlap、pca、sleeve 日收益 CSV）、
  `portfolios/`（4 × 全量 JSON + 日收益 CSV）、`summary.json`、`run_log.txt`。
