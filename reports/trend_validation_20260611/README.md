## Q2. 贝塔判定（综合 D1/D2/D3，逐族）

- **A 族 + C3：主体是贝塔。** short 大亏、2023 贡献 ≥98%（A3/C3 超 400%）、
  滚动窗口正占比 ≤67%、风险调整后接近或劣于 B&H。它们赚的是 2023 单边行情，
  此后三年在还回去。
- **B 族 + C1/C2：混合——显著贝塔成分 + 真实的择时价值。**
  贝塔证据：short 全亏、利润全在 long 侧、CI 含 0。
  技能证据：风险调整后 10–17× 优于 B&H；2023 占比 45–71%（4h 版 ≤59%）；
  滚动 80–100% 窗口为正；B&H 亏钱的 DOGE/LINK 上仍能盈利。
  **它们的价值不是"涨得比拿着多"，而是"以 1/10 的回撤拿到同量级收益"——
  这是择时的真实贡献，但它未能通过 V1 的集中度检验（见 Q6）。**
- 结论用词："纯贝塔"不成立（D2 风险调整证据明确），"已验证的技能"也不成立
  （V1 全灭 + D3 无显著性）。如实记录为：**未通过验证的混合体**。

## Q5. 总结论（如实，按检查清单第 5 项同等固化）

**经典趋势原型的毛利无法通过预注册的稳健性验证（0/15）。** 具体构成：
1. **V1 集中度是普适死因**：所有配置剔除 top 5% 交易后毛利转深负（−$3k~−$142k）
   ——利润完全集中在少数大赢家中。
2. 多数配置（11/15）同时死于时间稳健性；A/C3 族利润主体是 2023 牛市贝塔。
3. 无任何配置的单笔均值统计显著。
4. 但"主体为牛市贝塔"对 B/C-中速族**不成立**：风险调整后 10-17× 优于 B&H 是
   真实的择时贡献。精确的表述是：**有一簇值得继续研究的混合体，但以当前形态
   没有一个可以宣称稳健。**

## Q6. 观察节（本次不行动）

1. **V1 对趋势形态的结构性属性**：趋势跟踪的设计目标就是收割尾部大趋势，
   "剔除 top 5% 后毛利>0"对该类策略可能是结构性不可达（文献中 CTA 利润
   集中度本就极高）。本次按预注册 gate 如实判 FAIL、不放行；但下一阶段的
   方法论设计可考虑区分"脆弱集中"（top 交易来自单币单月）与"结构性尾部收割"
   （top 交易跨币种跨年份分散）——这是 gate 设计问题，留给下一阶段定义，
   并必须在跑之前预注册。
2. top 交易的分布特征本次未统计（top 5% 来自哪些币/年），是上述区分的
   第一个待做数据点。
3. V3 邻域 13/15 配置 100% 平坦——参数面健康，问题不在参数挑选。
4. V4 全过且 5 tick 诊断下头部配置仍 $40k+ ——滑点不是约束。
5. short 侧 15/15 全亏：本数据期做空趋势接近纯成本。"long-only 变体"是
   显而易见的下一个假设，但属于配置变更，本次按铁律不做不测。
6. B&H 的 funding −$55.7k（毛利的 46%）再次确认：本市场任何长持仓策略
   不计 funding 的回测数字都是虚的。
