## (Q2) 机制：更早入场是真趋势捕获、撞成本墙、还是 whipsaw 噪声？— **三者皆非，是"对称的快"**

**(3b) 早期捕获确实为真**（ADX 直觉在入场时点层面被证实）：对 V0 的 top-10% 大赢家，V_fast 在 V0 入场时
**已持同向仓**的比例——

| 样本 | V_fast 更早入场占比 | 中位提前 |
|------|---------------------|---------|
| OKX | **44/47 = 93.6%** | 30.0h |
| Binance | **78/84 = 92.9%** | 32.0h |

V_fast 确实在 ~93% 的大趋势里比 B2_4h 早进 ~30h（~7-8 bar）。

**(3c) 但毛利反而 ↓31/33%，不是 ↑** → **不是撞成本墙**（撞墙需毛利升、净利被费用吃平）。早进却毛利更低，
原因在 **(3a) "快"是对称的**：

| 样本 | 中位持仓 V0→V_fast | whipsaw(≤8h)占比 | whipsaw 净利 | 费用 V0→V_fast |
|------|--------------------|------------------|--------------|----------------|
| OKX | 196h → **76h** | 2.8% → **10.4%** | −$3.5k → **−$37.3k** | −$4.7k → −$13.4k |
| Binance | 192h → **76h** | 4.9% → **8.9%** | −$10.8k → **−$64.7k** | −$8.6k → −$23.5k |

更快的 EMA **早进也早出**：中位持仓从 ~195h 砍到 76h（~1/3），每个趋势只吃前段就被更快的反向交叉震出，
随后在回调里反复 whipsaw（whipsaw 笔数翻 ~3 倍、净亏翻 6-10 倍、whipsaw 胜率 0%）。胜率几乎不变
（28-29% 两版本一致）——不是赢得更少，而是**每个赢家吃得更短 + 输的 whipsaw 更多**。净效应：早进的好处
被早出 + churn 完全抵消并反超，**毛利反降**。

**结论（直接回应 ADX 引出的悬念）：** "70% 利润在早期入场"是关于 **B2_4h 自身哪些交叉最好**（趋势起点的），
**不等于"用更快的均线"**。更快的均线是**对称的**——无法只要"早进"而不带来"早出 + 更多 whipsaw"，后者
在两个样本上都主导。B2_4h 的慢均线**骑趋势更久（196h vs 76h）**，而骑得久正是钱所在。与裸基线"慢参数
胜出"完全一致。

## (Q3) 笔数 / 验证周期账（加法陷阱）

| 样本 | 笔数 V0→V_fast | 毛利 | 净利 | 验证周期 (PRIMARY) |
|------|----------------|------|------|--------------------|
| OKX | 470→1332 (+183%) | ↓31% | ↓46% | 9.1y → **26.8y** |
| Binance | 836→2334 (+179%) | ↓33% | ↓37% | 4.3y → **6.5y** |

**加法陷阱兑现**：多 ~3 倍笔数（事件密度），但日 M2M 年化 Sharpe 反降，验证周期不缩反而**延长**。
更多信号是更差的信号——加法不等于更快验证。

## (Q4) 判定 F1–F4（预注册，不可改）

| 门 | 判据 | OKX | Binance | 结果 |
|----|------|-----|---------|------|
| **F1 改善真实** | V_fast 净 Sharpe > V0 | False (0.65→0.38) | False (0.94→0.77) | ❌ FAIL |
| **F2 成本可活** | V_fast 净利 > V0 | False (−46%) | False (−37%) | ❌ FAIL |
| **F3 机制成立** | 毛利 > V0 且大赢家早入≥50% | False（毛利↓，早入93.6%✓） | False（毛利↓，早入92.9%✓） | ❌ FAIL |
| **F4 周期不恶化** | V_fast 周期 ≤ V0 | False (9.1→26.8y) | False (4.3→6.5y) | ❌ FAIL |
| | | | **ALL_PASS = False** | **更早入场无优势** |

四门全败双样本。注意 F3：早期捕获子句**通过**（93%），但毛利子句**失败**（毛利反降）——机制不是"假"，
而是"早进的真捕获被对称的早出+churn 抵消"。

## (Q5) 结论

**更早入场(EMA10/30)对 B2_4h 无优势，双样本每轴皆劣；B2_4h 维持原状(EMA20/100)。** 既非"真捕获更多"
（毛利反降）、亦非"撞成本墙"（撞墙需毛利升）、亦非"纯 whipsaw 噪声"（早期捕获 93% 为真）——而是
**"快"的对称性**：更快均线无法只要早进而不带来早出 + 更多 churn，后者双样本主导，B2_4h 慢均线骑趋势
更久(196h vs 76h)正是其优势来源。与裸基线"慢参数胜出"一致。

**与 funding / ADX 并列——已知靶子上的第三次调整，一致教训：**
- funding 确认：OKX 描线 / Binance 翻车 → 双样本不一致判死。
- ADX 过滤：双样本一致反向（删早期入场）→ 判死。
- 更早入场：双样本一致每轴皆劣（对称的快）→ 判死。

三次（指标过滤 ×2 + 参数调整 ×1）在已知靶子 B2_4h 上的增强尝试**全部无效**，且都被同一防线栈
（双样本 + 机制 + 周期）正确判死。**"更早入场应该更好"的悬念就此终结**——B2_4h 没有便宜的参数/指标
增强；趋势线维持资源关闭。安心转向正交新信号（期权 VRP 等离盒载体）。

## (Q6) 观察节
- 若未来仍想"更早 + 不早出"，需要**非对称**结构（快入场 + 慢出场，或趋势确认出场），但这引入新自由度、
  过拟合风险更高，且 B2_4h 已资源关闭，不在当前优先级。本研究的价值是排除"换更快均线对"这一最朴素、
  最易自欺的猜想。
- 本研究第三次验证防线栈按设计工作：它在 funding(描线)、ADX(反向)、faster-entry(对称劣化) 三种不同失败
  模式下都正确判死，无一被"OKX 单边好看"或"机制部分为真"骗过。
