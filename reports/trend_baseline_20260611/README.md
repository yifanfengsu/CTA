## 成本口径（原文，置顶）

> 入场与出场均按信号 bar 收盘价 ±1 tick 以 taker 成交（费率 0.05%/边）；
> 不假设任何 maker 成交；计入真实 OKX funding（8h 结算）；
> 无滑点压力测试（留下一阶段）；这是对趋势策略偏保守的成本口径。

补充口径：毛利 = 成交价（含 ±1 tick 不利滑点）PnL，费前、funding 前；每信号固定名义
$10,000（纯名义记账，无杠杆语义），整数张数按 ctVal 取整，单仓模型；聚合边界为 UTC
整 4h/整日（库内上海时间 → UTC−8h 显式换算；首根 1d bar 仅 8h 数据、末根不完整，
对指标预热影响可忽略）；funding 约定：OKX 正费率 = 多头付空头，结算时刻按
entry < t ≤ exit 计入，结算价取结算前一分钟 1m close。**funding 模块手算自检通过**
（3 个结算逐项比对，差值 0，见 `funding_module_selfcheck.md`）。

## 方法论定位（决定本研究的解释边界）

**无优化的全样本筛查**：15 个配置全部为文献经典原型、预先写死、零扫描零调参——
无拟合即无过拟合，全样本使用方法论上成立。但结论性质是"筛查"而非"验证"：
**PASS = 获得进入下一阶段（OOS/邻域/稳健性/集中度检验）的资格，不是"有 edge"**。
gate 预注册于脚本头部（GATE-1 毛利≤0 → FAIL；GATE-2 毛利>0 但每回合毛利<0.15%
名义 → MARGINAL），结果出来后未修改。

## Q3. 时间框架对比（4h vs 1d）

- **净利无系统性方向**：A1/B1/B2/C1 两框架接近或互有胜负（B1_4h $50k > B1_1d $38k；
  C1_1d $71k > C1_4h $64k）；唯 A2_1d ($30k) 显著强于 A2_4h ($16k)。
- 结构性差异：1d 笔数少 5–8 倍、单笔毛利厚（每回合毛利 1.3%–19.9% vs 4h 0.26%–3.0%）、
  手续费占比低一个量级；但 funding 按持仓时间计费，两框架量级相同（同为 $6.5k–$21.7k），
  故 **1d 的成本结构由 funding 主导（funding/手续费 ≈ 7–55×），4h 两者同量级**。
- 样本量上 4h 显著占优（216–1,222 笔 vs 30–526 笔），下一阶段的统计检验应以 4h 为主、
  1d 为确认。

## Q4. funding 的真实量级（历史研究从未有过的真数字）

| 量纲 | 数字 |
|---|---|
| 每配置 funding 合计 | −$6.5k ~ −$21.7k（全部为净支出——趋势持仓平均在"付费方"） |
| funding / 毛利 | 15 配置中位 **≈ 27%**；最重 **A3_1d 85%、C3_1d 82%、A1_1d 57%、C3_4h 56%** |
| funding vs 手续费（1d 配置） | funding 是手续费的 7–55 倍，**是 1d 趋势策略第一大成本** |
| 机制 | 趋势策略系统性持有"拥挤方向"（牛市做多正费率），carry 成本随持仓时长线性累积 |

结论：**任何不计 funding 的趋势回测在该市场都系统性高估净利**；持仓越长（A3/C3/1d）
被吃越狠——长周期配置的毛利厚度优势大部分被 funding 抵消。
