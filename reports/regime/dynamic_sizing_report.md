# MR-5m 动态仓位回测 — 任务C（C1-A → C2 → C3）

- 区间: 2023-01-01 → 2026-05-28（UTC），全部 5 币种
- 特征: **atr_ratio**（当根 Wilder ATR / 过去24根 ATR 均值），取其在过去 **N=200** 根中的滚动分位（在线/因果，无未来信息）
- **无路径依赖**：仓位不改变开/平仓决策，交易序列与基准完全相同，逐笔重算 size/PnL/费用
- 效率 eff = 净PnL / 总名义（bps）；Sharpe 为日 PnL 年化(√365)，仅作相对比较

**基准（$500 统一仓位）**：49,987 笔，总名义 $24,993,500，净 $343,860，PF 1.68，回撤 $6,559，Sharpe 7.10，效率 137.6bps。
