## TL;DR

**MR-5m FLAT 在真实数据上没有 edge。** 三个配置全亏：全期 PF 0.81-0.85、
test 期 PF 0.81-0.83；**毛利≈0（全期 +$6k~+$10k，$50M+ 成交额上是噪声），

## 第 3 步：差异归因

见 `demo_attribution.md`。核心：笔数反而 +19.4%（demo 不虚增机会，虚增
单笔质量）；单笔均值 +$8.88 → −$1.09（−$9.98/笔）；胜率 48.3% → 36.1%；
**demo 毛利 edge 的 ~99% 是数据假象**。

## 第 4 步：诚实结论

**(Q1) MR-5m FLAT 在 mainnet 上有 edge 吗？——无 edge。**
全期 PF 0.850（C2），test 期 PF 0.826；按任务判定参考落在"PF < 1.0 →
策略在真实数据上不盈利"。且这是**乐观口径**下的结果（入场 maker 挂单
100% 成交假设、出场仅 1 tick 滑点、未含资金费率）；真实执行只会更差。
更本质的是毛利≈0：**信号本身没有方向性信息，不是"成本略高于薄利"的边际
状态，没有可通过降本拯救的空间。**

**(Q2) 机制性结论是否重现？——基本不重现。**
- "2024 是全市场 MR 逆风年"：**不重现**。mainnet 上各年 PF 均匀地差
  （0.83-0.89），2023 反而最高；C1 下 2023 还是最差。"2024 特异性"是
  demo 数据的伪象。
- "whipsaw 是敌人、健康月 97%"：**前提不存在**。该叙事预设了"总体盈利 +
  少数失效月"，mainnet 上没有健康月份可言（无一币种-年份 PF>1），
  失效/健康的二分法失去对象。
- "DOGE 值得保留（PF 1.67 第二强）"：**不重现**。DOGE PF 0.873，只是
  五币中亏得最少之一；币种去留问题在基线无 edge 时无意义。

**(Q3) 口径 A vs B：差异可忽略；ATR filter 无法创造 edge。**
C2 与 C3 净利差 $957（1.4%）、PF 同为 0.850——阈值漂移问题在"基线本身
为负"面前无关紧要。filter 相对 C1 少亏 $37k，但那是"少交易 → 少交手续
费"，正是历史研究早已识别的 "looking better by trading less" 陷阱：
**过滤器能少亏，不能把毛利≈0 的信号变成正期望。**

**(Q4) 分币种：没有任何币种在真实数据上有 edge。**
全期 PF：SOL 0.886 > DOGE 0.873 > ETH 0.836 > LINK 0.827 > BTC 0.807。
注意排序与 demo 叙事几乎无关（demo 称 DOGE 第二强、BTC 干净——mainnet
上 BTC 反而最差）。差异在"亏多亏少"，不在"有无"。

**(Q5) 局限性：**
1. 未含资金费率（fade 策略常逆费率方向持仓，计入只会更差）；未做滑点
   压力测试（同向，只会更差）——两者都不改变结论方向，只会强化。
2. 入场假设 maker 限价单在信号 bar 收盘价 100% 成交，真实成交率 <100%
   且有逆向选择；此偏置同样是乐观方向。
3. demo 分币种 test 拆分未重算（禁用旧库），分币种归因仅定性。
4. 口径 B 的 p40 在 train 期内导出后用于全期回测，train 段存在轻度
   in-sample；鉴于 C3≈C2≈亏损，不影响任何结论。
5. 本任务只验 FLAT 基线；v2B/DYN 未跑（仓位放大器不能改变毛利符号，
   在基线无 edge 时大概率同样无效，但这句是推断不是实测）。
