# DYN-v2 设计与 walk-forward 验证 — 按"ATR 绝对水平"重做仓位系统

- 运行(UTC)见 `run_log.txt`（2026-06-09）。纯本地回测；实盘 runner / 基准引擎 / 任何参数 **均未改动**。
- 复用 `scripts/backtest_mr_5m_compare.py` 的指标/加载/合约规格；新脚本 `scripts/research_dyn_v2.py`。
- **无路径依赖**：sizing 不改变开/平仓决策，引擎每币只跑一次，逐笔重算 size/net（同 DYN-v1 口径）。
- 基线恒为 **FLAT $500**（干净零假设，回答"换轴是否真的更好"）。拆分同 ATR-filter OOS / atr_ratio 拆解：
  train=earliest 2/3（到 2025-04-09 08:00 UTC），test=latest 1/3（21,298 笔）。
- 名义口径：表中 notional = 实际合约名义（$500 保证金 × 5x ≈ $2,500），FLAT 均值 $2,489。

## 第 0 步 — 已有成果核查（关系：直接续作 atr_ratio 拆解）

- `atr_ratio_decomposition_20260609/`（**起点**）：DYN-v1 押错维度，atr_ratio 是 ATR 绝对水平的代理；
  真因子是绝对分位（5/5 币 PF 0.49→2.84）；v1 痛点格"低绝对水平×高 atr_ratio" PF 0.24 却被 v1 加注 $750。
- `atr_filter_oos_validation_20260609/`（**caveat 来源**）：组合 Max DD 由**大波动交易**制造，filter 碰不到；
  → 继承结论：**"按绝对水平加注"必然放大大波动暴露 → 必须显式量回撤**（本研究第 4 步硬门槛）。
- `dynamic_sizing_report.md`（**复用 C3 流程**）：DYN-v1 用 C3-1 时间序列 walk-forward（IS 2023–24 学切点 →
  OOS 2025–26 套用）、C3-2 邻近分位、C3-3 逐币 5/5。本研究第 3 步**复用同一流程**以保证可比。
- 无等价研究存在（v2 的"绝对水平分档 + 严格 gate"是首次），继续。

## 第 1 步 — 三候选（train 上拍定，无 leakage）

| 候选 | 设计 | 规则（按 train 期 ATR 经验分位） |
|---|---|---|
| v2A | 单维绝对分位（最保守） | p<40→$250 / [40,80)→$500 / ≥80→$750 |
| v2B | 二维 (绝对分位×atr_ratio) | p<40→$250；≥80→$750；中段 40–80：atr_ratio≥1.05→$500 否则 $250 |
| v2C | 绝对分位 + 低水平 skip（最激进） | p<30→**不做**；[30,60)→$250；[60,85)→$500；≥85→$750 |

## 第 3 步 — Walk-forward（C3 同流程：IS 2023–24 → OOS 2025–26）

| 候选 | IS 优势 | OOS 优势 | PF比(OOS/IS) | 邻近±5pct 均胜FLAT | 逐币 OOS |
|---|--:|--:|--:|:--:|:--:|
| v2A | +93.1% | +52.5% | 1.54 | ✓✓ | 5/5 |
| v2B | +100.8% | +53.6% | 1.53 | ✓✓ | 5/5 |
| v2C | +105.9% | +53.9% | 1.49 | ✓✓ | 5/5 |

- OOS 相对优势虽低于 IS，但**仍强正**（+53%），PF比 >0.85（实际 ~1.5，因 2024 逆风压低 IS PF）。
- 邻近分位（±5pct）OOS 仍胜 FLAT，逐币 **5/5** 全胜（v2B OOS：BTC/ETH/SOL/LINK/DOGE 全部 cand>flat）。
- **三候选均通过 walk-forward 全部子项。**

## 第 4 步 — 回撤显式度量（硬门槛）

| cfg | maxDD$ | maxDD/FLAT | DD/净利 | 最差单日$ |
|----|------:|:--:|------:|------:|
| FLAT | 2,250 | 1.00× | 0.0119 | −1,129 |
| v2A | 3,415 | 1.518× ✗ | 0.0118 | −1,752 |
| v2B | 3,426 | 1.523× ✗ | 0.0118 | −1,737 |
| v2C | 3,422 | 1.521× ✗ | 0.0120 | −1,730 |

- **(4c) 三候选全部不达 ≤1.5× 硬门槛**（实际 ~1.52×，仅微超）。
- **但关键诚实点**：**DD/净利 比率与 FLAT 完全相同（~0.0118）** → 绝对回撤变大**纯粹**因为收益等比放大，
  **风险调整后的回撤效率没有恶化**。这不是"PnL 涨 50% 但 DD 涨 100%"的劣化，而是同一风险收益比的等比放大。

### (4b) 协同行情压力（caveat 3 的直接度量）

- test 期内 5 币**同向**最大并发 = **5/5**（确实发生全市场同向）。
- 落在"≥4 币同向"窗口内的 cohort 交易 = 4,259 笔（占 test 20%）。
- cohort 段 PnL：FLAT +$5,483 / v2A +$13,757 / **v2B +$15,101** / v2C +$15,710。
- 即 v2 在协同行情下承载 **≈2.75× FLAT** 的暴露。此处为正（test 偏 MR 有利），但**风险对称**——
  若协同行情反向，v2 亏损也是 2.75×。**这正是 v2 回撤超额的来源，也是必须配组合风控的理由。**

## 第 5 步 — 组合风控前瞻（探索性，非正式设计）

在 test 上叠加两条简单规则（同向并发≤3；日亏 −2% 净值则停开新仓 24h）后重算：

| cfg | 无风控 net / DD | 有风控 net / DD | 有风控 DD/FLAT |
|----|------:|------:|:--:|
| FLAT | 189,227 / 2,250 | 160,598 / 1,843 | 1.00× |
| v2B | 290,270 / 3,426 | 247,812 / **2,684** | **1.456× ✓** |
| v2A | 288,314 / 3,415 | 245,305 / 2,713 | 1.472× ✓ |
| v2C | 285,625 / 3,422 | 243,269 / 2,670 | 1.449× ✓ |

- **简单组合风控把 v2B 的 DD/FLAT 从 1.52× 降到 1.46×（进入 ≤1.5× 门槛内）**，同时相对优势保留
  （v2B vs FLAT 有风控后仍 **+54%**）。→ 直接印证 finding 5：**绝对水平加注的回撤超额可被组合风控中和。**
- 说明：本步仅前瞻探索，组合风控正式设计是独立任务（计划书阶段 2）。

## 第 6 步 — 5 种诚实性检验

| 检验 | v2A | v2B | v2C |
|---|:--:|:--:|:--:|
| 1 参数稳健（±1pct，net 变化<10%） | ✓ | ✓ | ✓ |
| 2 子周期（前后两半都胜 FLAT） | ✓ | ✓ | ✓ |
| 3 单币种（5/5 胜 FLAT） | ✓ 5/5 | ✓ 5/5 | ✓ 5/5 |
| 4 平均名义 ≤1.1×FLAT | **✗ 1.113×** | ✓ 0.989× | ✓ 1.053× |
| 5 避开 v1 痛点格（527 笔，v1→$750） | ✓ →$250 | ✓ →$250 | ✓ →$250/skip |

- **v2A 失败检验 4**（靠 +11% 名义换 PnL）→ **作废**。
- **v2B / v2C 五项全过**；尤其 v2B 名义 0.989×FLAT（不靠加注），且把 527 笔 v1 痛点格从 $750 降到 $250。

## 第 7 步 — 诚实结论

### Q1 — 哪个候选通过全部 gate？
**严格地说：没有候选通过"全部"gate——三者都卡在 ≤1.5× 硬回撤门槛（~1.52×）。**
- **v2A**：另外还失败检验 4（名义膨胀）→ 直接作废。
- **v2B / v2C**：除该回撤门槛外，walk-forward 全过、5 种诚实性检验全过。
- 该回撤门槛的失败是**微超（1.52×）且为收益等比放大所致（DD/净利与 FLAT 相同）**，
  并可被组合风控中和（第 5 步 →1.46×）。

### Q2 — v2B（最优候选）vs FLAT
- 净利 **+53.4%**（189k→290k）；PF **+0.35**（2.06→2.41）；
- maxDD +52%（2,250→3,426，1.52×）**但 DD/净利不变（~0.0118）**；
- 平均名义 **0.989×FLAT**（不靠加杠杆）；逐币 **5/5**；walk-forward OOS 优势 +53.6%。

### Q3 — v2B vs DYN-v1 (C2-1)（次要）
同 test 集、同 FLAT 基线下：

| | net | Δ vs FLAT | PF | avg名义 | 效率(net/名义) |
|--|--:|--:|--:|--:|--:|
| FLAT | 189,227 | — | 2.06 | 2,489 | 76.0 |
| v1 (C2-1) | 228,167 | +20.6% | 2.36 | 2,011 | 113.5 |
| **v2B** | **290,270** | **+53.4%** | **2.41** | 2,462 | **117.9** |

- v2B 绝对净利与效率都**优于 v1**（+53% vs +20%；效率 117.9 vs 113.5），且修掉了 v1 的痛点格。
- （v1 的 DD 未在本轮按同一排序口径算，故不并列 DD；v1 净利较低，其有序 DD 应介于 FLAT 与 v2B 之间。）

### Q4 — 建议
**采纳"上线但需先做组合风控"——推荐候选 v2B，但不得单独上线。**

理由（保守、数据驱动）：
1. v2B 是强候选：换轴真实有效（+53% 净利、PF +0.35、**名义不增反降**、5/5 币、walk-forward 稳健、
   修掉 v1 痛点格），且优于 v1。
2. **唯一未达的是 ≤1.5× 硬回撤门槛**，且仅微超、源于收益等比放大（风险调整 DD 未恶化）。
3. 但第 4b 表明回撤超额来自**协同行情 2.75× 放大暴露**——这是真实的相关性回撤风险，不能无视。
4. 第 5 步前瞻显示**先上组合风控**可把 v2B 的 DD 压回门槛内（1.46×）并保留 +54% 优势。

**落地路径：** 计划书阶段 2 的**组合层风控先完成并验证** → 然后用相同 gate（含 ≤1.5× DD）复测 v2B →
通过后再作为 B 账号实验上线。**v2A 作废**（名义膨胀 + DD）；**v2C 可行但无优于 v2B 的理由**且会跳过交易（牺牲样本/稳健），不推荐。

> 不硬塞结论：若不接受"先做组合风控"这一前置，则当前**所有候选都不达 ≤1.5× 回撤门槛**，
> 严格结论为"不推荐立即升级"。本报告不把任一候选标记为"可直接上线"。
