## 第 1 步 — 规则定义与配置
- R1 日亏熔断：当日累计净 PnL ≤ −X%×当日初始 equity → 停新仓 N 小时。X∈{1,1.5,2,3}%，N∈{12,24}h（8 个）。
- R2 同向上限：开新仓时同向并发持仓 ≥ K 则拒绝。K∈{2,3,4}（3 个）。
- R3 横截面降档：同 5m bar 同向信号 ≥ M → D1 全跳过 / D2 全降 small / D3 按量降（m=3 降1档,4 降2档,≥5 跳过）。
  M∈{3,4,5}×{D1,D2,D3}（9 个）。
- 共 20 配置 × {FLAT, v2B} = 40 次回测。

### 单规则"最优"判定（严格按 双重条件：ΔPnL>−3% **且** 降 DD）

- **R1：无有效最优。** 唯一能把 DD 压到 ≤1.5× 的是 R1（X1.0_N24h→0.991×），但**所有** R1 配置
  ΔPnL ≤ −11%（违反 >−3%），且误伤额 +$33k~+$45k。→ **R1 在本数据下无效（且为误伤式），不进入 2.B。**
- **R2：无有效最优。** 不伤 PnL 的阈值（K3/K4）**maxDD 完全不变（3,426 = 1.522×）**；更激进的 K2
  反而 DD 更差（3,433）且误伤 +$27k。→ **R2 不能在不伤 PnL 的前提下降 DD。**（K4 误伤极小，但降 DD=$0。）
- **R3：无有效最优。** **所有 9 个配置 maxDD 均不变（3,426 = 1.522×）**。R3 最干净（M4_D1 误伤 −$82），
  但对 maxDD 毫无作用。→ **R3 不能降 maxDD。**

> **核心事实：没有任何单规则能在 ΔPnL>−3% 的前提下把 v2B 的 maxDD 降到 ≤1.5×。**
> R1 能降 DD 但靠"少交易"误伤 11~15% 利润；R2/R3 在安全阈值下**根本不动 maxDD**。

### 普适性（FLAT 线）
- R1 在 FLAT 同样误伤（最温和 X3.0_N24h 仍 −10.2%，误伤 +$19,307）→ 普适失败。
- R2 K4：FLAT ΔPnL −0.1%，maxDD 不变（2,250）→ 普适地"无害但不降 DD"。
- R3 M4_D1：FLAT ΔPnL +0.2%，maxDD 不变，误伤 −$387 → 普适地"干净但不降 DD"。
- 三规则在 FLAT 与 v2B 上结论一致：**效果（或无效）是普适的，非 v2B 条件性。**

### 实施成本
- R1：需日界 equity 快照 + 当日 PnL 实时累加 + halt 计时器（中等状态）。**但已判无效，不实施。**
- R2：开仓前查同向并发计数（极轻，已有持仓表即可）。改动小。
- R3：同 5m bar 内同向信号计数后再下单（需把同 bar 多币信号收齐再决策，runner 改动中等）。

## 第 4 步 — 反应式熔断辨析（详见 `reactive_breaker_comparison.md`）
- **R1 = 反应式熔断死胡同同型**：PnL 触发 + 停新仓 + 大额正误伤 + 复现 74 配置全负模式 → 放弃。
- R2 / R3 = 事前结构性触发（并发数 / 信号广度），非 PnL 反应，误伤≈0 → **非**该陷阱。

## 第 5 步 — 诚实结论

### Q1 — 三条规则各自有效吗？
- **没有一条**能在 v2B 上把 maxDD 降到 ≤1.5×FLAT 同时 ΔPnL>−3%。
- **R1 是误伤式熔断，应放弃**（降 DD 仅靠砍 11~15% 利润，误伤 +$33k~+$45k；机制同历史死胡同）。
- **R2、R3 在安全阈值下根本不降 maxDD**（DD 不变 3,426）；它们只是廉价地略减"协同事件暴露"。
- 普适 vs 条件性：三规则结论在 FLAT 与 v2B 上**一致（普适）**——R1 普适误伤，R2/R3 普适地"不动 DD"。

### Q2 — 最优阈值
- R1：**无**（最优也违反 PnL 条件）→ 规则放弃。
- R2：K4 是"最不伤"的（误伤 $146、ΔPnL −0.1%），但**降 DD = $0**，不构成有效最优。
- R3：M4_D1 是"最干净"的（误伤 −$82、ΔPnL +0.0%），但**降 DD = $0**，不构成有效最优。

### Q3 — 是否有规则与反应式熔断死胡同同型？
**有：R1。** 机制同型且复现失败模式 → **必须放弃，不进入阶段 2.B。**

### Q4 — 给阶段 2.B 的建议
1. **R1 放弃**（反应式熔断陷阱）。
2. **R2(K4) / R3(M4_D1 或 M4_D3)** 可作为**近乎零成本的结构性护栏**进入 2.B —— 它们以 ~0 PnL 代价、
   ~0 误伤减少**协同同向暴露**（前瞻尾部风险），但**必须明确：它们不降本窗口的 maxDD**，不是 v2B
   过 1.5× 门槛的解。
3. **给 2.B 的最重要警示（数据驱动）**：v2B 的 maxDD **不是协同相关性问题**——5/5 协同事件净为正且很小，
   maxDD 由**大波动单笔交易**主导（与 ATR-filter OOS 结论一致）。因此 **R2+R3 组合大概率仍无法把 maxDD
   压到 ≤1.5×**。2.B 应：(a) 仍测组合但**预期偏低**；(b) 显式重审"单条实现路径上的绝对 1.5× DD"是否为
   合适门槛——dyn_v2 已证 v2B 的 **DD/净利与 FLAT 相同**（回撤随收益等比放大，风险调整未恶化）；
   (c) 真正能降 maxDD 的杠杆可能是**极端 ATR 处的单笔 size 上限**（本阶段范围外，建议作为后续方向）。
4. 不硬塞：本阶段三规则**没有一条**满足"降 v2B DD ≤1.5× 且不伤 PnL"，故**不推荐**任何一条作为 v2B 的
   DD 解决方案。R2/R3 仅以"廉价护栏"身份进入 2.B 备选，R1 直接出局。
