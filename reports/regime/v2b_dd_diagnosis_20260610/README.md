## TL;DR

**元凶已识别，但它不是市场模式——是数据异常。**
2025-05-29 max DD 窗口的 16 笔交易里，前 12 笔合计仅 −$10.33；真正的伤害是
15:54–16:14 的 4 笔（−$3,415.23，占 DD 的 99.7%）。这 4 笔全部 fade 了 SOL/DOGE
当时数据里的**合成直线 ramp**（每分钟精确 ±10.3 等步长的阶梯式闪崩+闪涨，单根 K 线瞬间
复原，BTC/ETH/LINK 同时刻纹丝不动，真实市场无任何对应记录）。全期扫描发现**同类异常事件
共 598 次**（SOL 344 / DOGE 216 / ETH 23 / LINK 15 / **BTC 0**），且这些异常窗口平时在
"送钱"——与异常重叠的交易贡献了 v2B test 净利的 **+$91,819（31.6%）**。2025-05-29 只是
该模式历史上对策略最不利的一次实现。**v2B 的 1.523× DD 和大盘净利数字都建立在被污染的
数据上。**（Q4 选项：A/B/C 之外的 **D：数据 artifact**——硬选三者之一会误导。）

**判读（含诚实边界）**：
1. 该模式不仅可复现，而且**高频**——平均每两天一次。它不是市场模式，是数据异常类

   均无真实市场对应；BTC 0 次、SOL 344 次的币种不对称也与真实跨币种传染不符）。
2. **方向上它通常对策略有利**：V 形 ramp 瞬间复原 → fade 入场拿到 midline 止盈。
   2025-05-29 是该类事件唯一一次"双向都先打穿止损再复原"的最不利实现。
3. 注意因果边界：±(5min/30min) 窗口重叠 ≠ 全额归因；检测器也会命中**真实**闪崩
   （如 2025-10-10/11 真实暴跌日在列）。污染量级的诚实区间是
   **11.5%（仅等差阶梯签名）～ 31.6%（全部检出事件）** 的 v2B test 净利。
4. 数据来源（推测，未定论）：下载脚本 REST 路径走 OKX 生产 history-candles，但项目
   `OKX_SERVER=DEMO`，下载器存在 gateway 路径（`--source gateway/auto`，DEMO 网关带
   `x-simulated-trading: 1`）与 gap 修复路径——部分区段可能取到 OKX 模拟盘合成行情。
   BTC 干净、SOL/DOGE 重灾的分布与"按币种/区段混入"一致。**确证需查下载 manifest，超出本诊断范围。**

## 第 5 步：诚实结论

**(Q1) 那天最显著的市场特征？**
不是市场特征，是数据特征：SOL/DOGE 在 15:52–17:00 出现等差直线阶梯式闪崩+闪涨
（±50%，单根复原，往返两次），同时 BTC/ETH/LINK 无事、5 币相关性低于平时（0.18 vs 0.33）。

**(Q2) 16 笔的核心共性？**
没有单一覆盖 16 笔的共性——因为 16 笔是两组事件：12 笔普通交易（净 −$10.33，正常噪声）
+ 4 笔在同一个合成 ramp 异常里双向被收割的交易（−$3,415.23）。后者的共性精确成立：
都 fade 了合成 ramp、都 1 bar 被止损、都因 ramp 推高 ATR 至 p89–p99 而被 v2B 给到 large 档、
价格都在止损后立即"回归"。v2B 放大了伤害（1.523×）但不是根因。

**(Q3) 模式可识别吗？可复现吗？**
高度可识别（等差步长、瞬间复原、币种不对称），高度可复现（全期 598 次）。但它是
**数据异常模式**，不是可供策略学习/规避的市场模式。

**(Q4) 综合判断：A / B / C 之外的 D——数据 artifact。**
- 不选 A：它不是"市场模式"，v2B 的弱点暴露于假数据而非真市场状态。
- 不选 B：它不是已知市场风险的极端版本——协同行情与极端 ATR 单笔此前都已被排除，本次确认
  连"波动率放大期"也只是 ramp 的伴生症状。
- 不选 C：共性极其清晰，不是"不幸的一天"。
- 按任务要求"不硬选粉饰"：**D. 2025-05-29 事件由合成/损坏的行情数据造成**，是高频数据异常类
  （598 次）中对策略最不利的一次实现。该结论同时意味着：基于本数据库的 max DD 与净利数字
  对真实市场的外推效力受损（见 Q5）。

**(Q5) 认知更新（非行动建议）**
之前不知道、现在知道了：
1. v2B 的 max DD（1.523× FLAT）**不度量真实市场风险**——分子由合成数据事件主导（99.7%）。
2. 对称地，**净利也被同一异常类抬高**：v2B test 净利的 11.5%–31.6% 与异常窗口重叠。
   污染是双向的，且净方向为正（异常平时送钱）。
3. 数据质量按币种分层：BTC 干净（0 事件），ETH/LINK 轻度（23/15），SOL/DOGE 重度（344/216）。
   既往所有以 SOL/DOGE 为主要利润来源的结论（含币种取舍、动态仓位 walk-forward、ATR filter OOS）
   的**效应量**都带有未知比例的异常贡献。
4. 对"是否接受 v2B 1.523× DD"判断的影响：这个比值的分子分母都来自被污染的数据，
   **该判断在当前数据上无法给出对真实市场有意义的答案**——无论接受还是拒绝都是对噪声表态。
   是否/如何处理数据并重新评估，是诊断之后的独立决策，不属于本研究。
