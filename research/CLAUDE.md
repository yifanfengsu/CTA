# research/ — 研究域宪法

> 本目录为研究域（前置研究/回测/策略验证脚本与产物）。宪法先立、脚本后迁
> （重构步骤 2 迁入 `research_*` / `backtest_*` 等脚本；本步骤目录暂空）。
> 顶层 CLAUDE.md 的铁律 A/B/C 与工作原则是本宪法的上位法。

## 1. 全部研究走流水线

任何新研究**必须**按 `.claude/skills/PIPELINE.md` 的九站走完一生
（第 0 站出生地检查 → 第 8 站尸检入库），站间交接物齐备。跳站 = 程序缺陷，
报告须显式声明缺了哪站及理由。

## 2. skill 调用义务（到站必用，不重新发明）

| 情形 | 必用 skill |
|---|---|
| 开题 / 设计 gate / 测多配置 | `preregistration/` |
| "检测到 X 个信号/对子/事件"类计数结论 | `noise-calibration/` |
| 配对/价差方向 | `cointegration/` |
| 横截面因子方向 | `ic-analysis/` |
| 误差棒/下界/验证周期 | `bootstrap-inference/` |
| 从 N 配置选最优出数字 | `multiple-testing/` |
| 生死判定 | `honest-verdict/` |
| 冻结数字核算 / 重大结论复核 | `audit-independent/` |

skill 已含实现（`.py` 均经冒烟自检），**不重复造轮子**；发现 skill 不足 →
先修 skill（另行 commit）再用，不绕开。

## 3. 预注册先于结果（铁律 A）

- gate/阈值/宇宙/OOS 纪律看结果前写死，预注册文档**先于结果单独 commit**
  （模板 `.claude/skills/preregistration/gate_template.md`）。
- 预注册 gate 判死后不得重定义复活；字面 gate 与综合判定分歧 → 并列报告、
  分歧方向必须朝保守。
- trial-ledger 全程记账（中途新增配置也入账），产物落盘须可还原全配置 Sharpe。

## 4. 双样本纪律

- 定义样本（OKX mainnet）+ 独立确认样本（Binance 2020–2026，含 2021 牛/2022 深熊）。
- **已知靶子上单样本改善的证据权重 ≈ 0**（描线先验，已被骗 5 次：funding/ADX/
  faster-entry/V1/vol-targeting）；"总收益提升"自动触发最高警惕。
- 机制证据在样本间/粒度间**翻号即否**，业绩数字再好看也判死。

## 5. 判死判活同标准

- 否定性结论与肯定性结论同等证据标准、同等固化（进 PROJECT_GUIDE
  "已验证的核心事实"）。
- 幸存者/方向偏差的方向显式标注（偏乐观样本判死 ⇒ 否定更稳）。
- 立项算术前置（铁律 B）：验证周期超资源窗口不立项，无论形态多有吸引力。

## 6. 报告结构规范

每份研究 README 按既有成熟范式（见 pairs / factor_scale / vol-targeting 报告）：

1. 头部：日期(UTC)/脚本/数据源/**回测假设显式**（费率/滑点/成交/funding）。
2. 置顶：定位与纪律声明（frozen 不删改）+ 机制/先验/样本偏差三段声明。
3. 预注册段（或指针到独立 PREREGISTRATION.md + 其 commit hash）。
4. 判定速览（置顶表格：gate 逐条 + 判定）。
5. **Q1–Qn 正文**（每问一节，数字进表格）。
6. 观察节（可复用教训，不设新 gate）。
7. 产物清单 + 复现命令。
8. **文档更新清单** + **"考虑过但未改动的文档及原因"**（显式列出，避免隐性决策）。
9. 结论行（最后一行：完成时间/关键数字/判定/push hash）。

## 7. 措辞纪律

- **资源关闭 ≠ 证伪**：不得称资源性关闭的信号"被证伪"或"差一点成功"
  （B2_4h 条款）。
- **事后移动球门写"程序污点"，不写"自我修正"**（V1′ 条款）。
- 数据不支持就写不支持，不强行给"积极"结论；答案条件于假设，conditional
  条款逐条写。

## 8. 边界

- 不修改归档实盘脚本与基准引擎；研究产出独立脚本 + markdown 报告 + 中间
  jsonl/json。
- 不碰 forward 域资产（见 `docs/AGENTS.md`）；数据问题移交 data_engineering 域；
  判活策略的冻结数字移交 audit 域独立核对。
