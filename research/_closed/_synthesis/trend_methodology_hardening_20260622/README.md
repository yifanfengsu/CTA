## Q3. Deflated Sharpe Ratio（Bailey & López de Prado 2014）+ 计算输入

**方法预定死（零挑选）**：主 = DSR（按原文，不自创变体）；辅 = stationary bootstrap 估
诚实 SE。绝不测多个打折方法挑一个。

**DSR 公式（全部 DAILY 单位）**：
```
PSR(SR*) = Φ[ (SR̂ − SR*)·√(n−1) / √(1 − skew·SR̂ + ((kurt−1)/4)·SR̂²) ]
SR*₀     = √V · [ (1−γ)·Φ⁻¹(1 − 1/N) + γ·Φ⁻¹(1 − 1/(N·e)) ]
DSR      = PSR(SR*₀)         deflated(haircut) Sharpe := SR̂ − SR*₀
```

**计算输入（B2_4h，看结果前定死的口径，偏度/峰度用实际分布，绝不假设正态）**：

| 输入 | 值 |
|---|---|
| 观测 Sharpe（日） SR̂ | 0.03428（年化 0.6548） |
| 日收益偏度 skew | **0.191**（实际，非正态假设） |
| 日收益峰度 kurt（非超额，正态=3） | **7.50**（超额 +4.5，肥尾） |
| 样本长度 n | 1,228 日（B2_4h own-live 3.36 年） |
| Var(SR across trials)（日，ddof1） | 1.313e-4（15 配置年化 Sharpe std 0.219） |
| γ（Euler-Mascheroni） | 0.5772156649 |

**DSR 结果（三 N 口径，sanity check 全过）**：

| N 口径 | N | SR*₀(年化) | **deflated Sharpe(年化)** | DSR 概率 |
|---|---:|---:|---:|---:|
| **有效 N（主）** | 2.35 | 0.145 | **0.655 → 0.510** | 0.826 |
| 名义 N | 15 | 0.388 | 0.655 → 0.267 | 0.688 |
| 扩展 N（下界） | 47 | 0.493 | 0.655 → 0.162 | 0.617 |
| 参照：PSR(0)（不打折） | — | 0 | 0.655 | 0.886 |

**bootstrap 诚实 SE（年化 Sharpe）**：iid 0.542 / block(L=8d，主) 0.530（见 Q4）。

- **sanity check 全过**：① 有效 N = 2.35 > 1 ✓；② SR*₀ ≥ 0 三口径全真 ✓；
  ③ deflated（0.510/0.267/0.162）**全部 < 观测 0.655** ✓——未违反 DSR 数学性质，无计算错误。
- **诚实读数**：去偏幅度对 N 口径高度敏感。主算（有效 N）haircut 仅 −22%
  （0.655→0.510），因为 15 个趋势配置只有 ≈2.35 个独立——"best-of-2.35"的运气门槛
  远低于"best-of-15"。名义/扩展口径 haircut 重得多（→0.267/0.162）。**三口径下
  deflated Sharpe 均保持为正、DSR 概率均 > 0.5，但无一达显著（< 0.95）。**

## Q4. 验证周期（从 bootstrap SE 反推）vs 原 15 年（iid）——两口径各贡献多少

**方法（修正自相矛盾）**：不用 (1.96/Sharpe)² 的 iid 闭式（它假设 iid，与用 bootstrap
纠正自相关自相矛盾）。改为基于 bootstrap SE 反推 time-to-significance：t-stat ∝ √T →
T* = T₀·(1.96/t₀)²，t₀ = SR_年化 / SE_年化。

**先做自相关诚实性检查（关键的反向发现）**：
- stationary bootstrap（Politis-Romano，均块长 L = round(中位持仓 196h /24) = **8 日**，
  B=10,000，seed 20260622）SE = 0.530；iid（L=1）SE = 0.542。
  **inflation = SE_block/SE_iid = 0.979（≈1，甚至略 <1）。**
- 直接佐证：B2_4h 日收益 **ACF lag1..5 = [−0.05, 0.02, −0.04, 0.02, 0.03]，全部≈0**。
- **诚实结论（数据裁决，反向于任务假设）**：任务预期"196h 持仓 → 日收益自相关 → block
  bootstrap 会拉长验证周期"。**实测日收益几乎无自相关**——因为日 M2M 收益 ≈ 持仓 ×
  日价格增量，持仓在持有期内恒定不制造收益自相关（增量近 i.i.d.）。**自相关修正
  在此处不重要**；验证周期的拉长几乎全部来自选择偏差打折，不是自相关。我按预注册
  做了 block bootstrap，结果显示它不重要——如实报告，不强行制造一个不存在的效应。

**验证周期对照（T₀ = 3.36 年）**：

| 口径 | 年化 Sharpe | SE | **验证周期** | 说明 |
|---|---:|---:|---:|---|
| 原方法锚点（组合 0.5 + iid） | 0.50 | — | **15.4 年** | (1.96/0.5)²，原"15 年"出处 |
| iid 闭式（B2_4h 自身 raw 0.655） | 0.655 | — | **9.0 年** | B2_4h 自身 raw Sharpe 比组合 0.5 高 → iid 周期反而更短 |
| block boot，raw | 0.655 | 0.530 | **8.5 年** | 自相关修正后 ×0.96（几乎无变化） |
| **block boot，deflated（有效N）** | 0.510 | 0.530 | **14.0 年** | 主算口径 |
| block boot，deflated（名义N） | 0.267 | 0.530 | 50.7 年 | |
| block boot，deflated（扩展N下界） | 0.162 | 0.530 | 138.4 年 | |

**分解（有效 N 口径，从 raw 8.5y → deflated 14.0y）**：
- **自相关效应：×0.96**（几乎为 1——见上，ACF≈0，反向于任务预期，如实记录）。
- **打折效应：×1.65**（=(0.655/0.510)²；名义 N 为 ×6.0，扩展为 ×16）。
- **拉长几乎全部来自选择偏差打折，非自相关。**

**对原"15 年"的诚实校准**：原 15.4 年用的是**组合 Sharpe 0.5 + iid + 未打折**。
B2_4h 自身 raw Sharpe 更高（0.655）→ 若按原 iid 法只 9 年。但去偏后回到 14 年（有效 N）
~ 51 年（名义）~ 138 年（扩展）。**原"15 年"数值上落在合理区间，但方法是错的**——
它用了较低的组合 Sharpe，恰好抵消了缺失的打折；honest 的 B2_4h 周期是 **≥14 年**
（最宽松口径），更严口径 51–138 年。无论哪个口径，**都远超项目可接受的资源窗口**。

## Q7. 观察节

1. **有效 N 是这次最重要的杠杆**：DSR 对"算多少个独立试验"高度敏感（haircut 0.510 vs
   0.267 vs 0.162）。趋势族高度相关（ENB 2.35）使 best-of-N 门槛远低于 naive N=15——
   这对**任何**未来"从相关配置族选最优"的研究都适用：必须报有效 N，否则要么过度惩罚
   （用名义 N）要么不惩罚（用幸存者 Sharpe）。
2. **自相关在日尺度上对这类策略不重要**（ACF≈0），这是一个可复用的方法发现：4h/1d
   趋势策略的日 M2M 收益近 i.i.d.，验证周期的 iid 闭式在**自相关维度**上其实够用——
   真正的偏差源是**选择偏差**，不是序列依赖。（铁律 B 强调打折 > 自相关修正，此处实测印证。）
3. **右偏活在逐笔尺度，不在日尺度**：日 Sharpe/Sortino 都在日尺度，已稀释了逐笔偏度
   9.13。若未来要给正偏策略一把真正公平的尺子，应考虑**逐笔/持仓尺度**的尾部指标，
   而非仅日收益下行调整——这是铁律 C 的一个待深化方向（本次仅描述，未立项）。
4. **本次零参数搜索、零策略改动、零 gate 改动**：全部口径写死于脚本 docstring，
   DSR/bootstrap/有效 N 按文献标准实现，三 N 口径与多 L 块长全部并列报告，
   不在多方法/多口径里挑结果。
